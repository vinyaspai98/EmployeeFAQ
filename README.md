
## Quick start

- Open teminal 1

- Install dependencies: `npm install` or `yarn`

- Start the front end server: `npm start` or `yarn start`

- Views are on: `localhost:3000`

- Open terminal 2

- Cd server

- Install dependencies: `npm install` or `yarn`

- Start the Back end server: `node index.js`
